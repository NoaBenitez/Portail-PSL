Documentation - Export Factures

Mode d'emploi complet pour l'export et le renommage des factures

⚙
Installation (première fois uniquement) Ces 6 étapes ne sont à faire qu'une seule fois pour configurer votre poste
1 Télécharger PDF24

Se rendre sur tools.pdf24.org/fr/creator et cliquer sur "PDF24 Creator Installer" pour lancer le téléchargement.

2 Installer PDF24

Ouvrir le fichier .exe téléchargé et cliquer sur Suivant à chaque étape jusqu'à la fin de l'installation.

3 Ouvrir PDF24

Une fois l'installation terminée, taper "PDF24" dans la barre de recherche Windows et ouvrir l'application.

4 Ouvrir l'assistant d'impression PDF

Dans PDF24, cliquer sur "Ouvrir l'assistant d'impression PDF".

5 Accéder aux paramètres

Cliquer sur "Paramètres".

6 Configurer l'imprimante PDF

Cliquer sur "Imprimante PDF", vérifier que les cases cochées correspondent à l'image ci-dessous, sélectionner le dossier de destination des PDF, puis cliquer sur "Appliquer". Fermer PDF24 ensuite.

⚡
Utilisation (à chaque export) Suivez ces étapes à chaque fois que vous souhaitez exporter des factures
7 Imprimer depuis Sage

Ouvrir Sage, aller dans Pièces clients → Factures. Renseigner les dates souhaitées (important), sélectionner toutes les factures puis cliquer sur "Imprimante".

8 Sélectionner PDF24

Dans la liste des imprimantes, sélectionner "PDF24".

9 Options avancées

Cliquer sur "Option avancée" et cocher la case "Générer une impression par document".

10 Modifier la mise à l'échelle

Dans Standard, cliquer sur l'icône paramètres puis sur "Avancé...". Modifier la mise à l'échelle : remplacer 100% par 82%, puis cliquer sur OK.

11 Attendre la génération

Ne pas fermer Sage ni les autres fenêtres qui s'ouvrent. Les PDF apparaîtront dans le répertoire choisi après quelques minutes.

12 Renommer avec l'outil

Ouvrir l'outil Renommage PDF, glisser toutes les factures dedans, cliquer sur "Analyser", attendre quelques minutes, puis descendre et cliquer sur "Télécharger les PDF renommés". Pensez à vérifier le nombre de PDF tout au long du process. Vérifiez également les noms des fichiers : il est possible que 2 ou 3 PDF ne soient pas renommés automatiquement, il faudra les renommer manuellement.

Documentation - Plannificateur

Mode d'emploi pour planifier et optimiser vos tournées

1 Importer un fichier Excel

Cliquer sur "Choisir un fichier" et sélectionner un fichier Excel (.xls ou .xlsx).

2 Préparer le fichier Excel

Le fichier doit suivre au mieux le modèle de colonnes ci-dessous. La date min doit être au minimum 2 mois avant la date du jour et la date max au minimum 2 mois après. La colonne Temps correspond au temps d'intervention. Chaque ligne représente un site, données brutes.

3 Vérifier la correspondance des colonnes

Vérifier que toutes les colonnes sont correctement reconnues. Si ce n'est pas le cas, modifier manuellement les correspondances.

4 Confirmer et géocoder

Cliquer sur "Confirmer le mappage" pour valider les colonnes, puis cliquer sur "Valider et géocoder".

5 Vérifier le géocodage

Attendre la fin du géocodage. Modifier les coordonnées GPS si des erreurs apparaissent, sinon cliquer sur "Continuer vers l'optimisation".

6 Configurer les règles d'optimisation

Définir les règles souhaitées ou laisser par défaut. La durée max par tournée correspond à la durée d'une journée (trajet + temps sur site). Le découpage est le nombre de sites par tournée ("Auto" recommandé). L'IA est également recommandée. Puis cliquer sur "Lancer l'optimisation".

7 Exploiter les résultats

Attendre la fin de l'optimisation. Toutes les tournées sont créées. Fonctionnalités disponibles :

• Editer : modifier une tournée
• Excel : exporter les tournées en fichier Excel
• Onglet Carte : visualiser les tournées sur la carte
• Onglet Analyse IA : remarques et conseils pour optimiser davantage
• Onglet Planning : suivre les dates auxquelles les tournées sont positionnées
• Onglet Coût : calculer les coûts de trajet et d'hôtel pour toutes les tournées

Documentation - Mise en forme du mémoire technique

Procédure complète pour transformer le contenu ChatGPT en document mis en forme via Figma.

1 Générer le HTML depuis ChatGPT

Insérer le Word brut du mémoire technique (juste le texte) et écrire "mets le en forme", puis cliquer sur le fichier .html pour le télécharger.

2 Ouvrir le plugin Figma

Ouvrir Figma via ce lien, c'est plus simple : html.to.design sur Figma, puis cliquer sur "essayez dans...".

3 Sélectionner le bon plugin

Sélectionner html.to.design.

4 Lancer le plugin

Cliquer sur "Executer".

5 Aller dans l'onglet File

Cliquer sur l'onglet "File".

6 Importer le HTML

Cliquer sur "Browse" et sélectionner le fichier HTML téléchargé depuis ChatGPT.

7 Vérifier les options

Vérifier que ces cases sont cochées puis cliquer sur "Proceed".

8 Confirmer

Cliquer sur "Ok".

9 Fermer la fenêtre du plugin

Le document s'est chargé en arrière-plan, fermer la petite fenêtre html.to.design.

10 Exporter les sous-pages en PDF

Sélectionner uniquement chaque sous-page (Ctrl + clic sur toutes les pages), puis descendre dans le menu de droite, sélectionner PDF, cocher la case "Inclure la zone de délimitation", fermer le popup puis cliquer sur "Exporter les X calques".

11 Fusionner les PDF

Ouvrir le .zip téléchargé et mettre tous les PDF qu'il contient sur ce site : ilovepdf.com/fr/fusionner_pdf, puis cliquer sur fusionner.
